# Neural Network Digit Visualization

Interactive visualization of a simple neural network for handwritten digit recognition.

## Objective

This project was created for educational and presentation purposes, aiming to demonstrate in a visual and intuitive way how a neural network processes handwritten digits and produces probabilistic outputs.

The application allows the user to:
- draw digits on a canvas
- visualize prediction probabilities in real time
- observe neural-network-based inference behavior

## Technologies

- HTML
- CSS
- JavaScript
- Canvas API
